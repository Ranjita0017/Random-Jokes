# Joke Hub
A Simple and beginner-friendly API project. <br>
A web application that fetches random jokes from the JokeAPI and displays them dynamically on the webpage.<br>

## Features
- Fetches random jokes using the JokeAPI.
- Supports both single-line and two-part jokes.
- Generates a new joke on every button click.
- Updates content dynamically using JavaScript.
- Uses Axios for API requests.
- Includes basic error handling for failed requests

## Technologies Used
- HTML
- CSS
- JavaScript
- Axios
- JokeAPI

## Learning Objectives
- Working with APIs using JavaScript.
- Using async/await for asynchronous operations.
- Handling API responses dynamically.
- Manipulating the DOM with JavaScript.
- Implementing basic error handling.
